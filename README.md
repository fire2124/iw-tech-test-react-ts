## 🏁 Getting Started <a name = "getting_started"></a>

👷 Running Locally

- `git clone https://github.com/fire2124/iw-tech-test-react-ts`
- `cd Frontend/src `

👷 Docker

- `docker build . -t dockerized-react`
- `docker run -p 3000:3000 -d dockerized-react`

👷 Installing

- Install: `yarn install`

👷 Available Scripts

- Build it: `yarn build`
- Run it: `yarn start`
- Test it: `yarn test`
